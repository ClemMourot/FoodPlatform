USER GUIDE :

TO USE THE APP LOCALLY :
  - clone the repository and set up your virtual environnement
  - install PostgreSQL
  - install dependencies by running "pip install -r requirements.txt" (install pip in case you haven't already)
  - in settings.py, modify the databases infos with your own database name, user and password
  - in shell, call the get_products and get_categories methods to collect data
  - run "python manage.py runserver" in the command line and click on the link generated


WEB USER GUIDE :
  - sign in or create your account by clicking on the person shaped logo
  - search for products in the search bars
  - you will then be asked to choose which product you'd like to find a substitute for
  - select a product (by clicking on "Choisir ce produit)
  - a list of potential substitutes will be displayed, you can access them by clicking on their names or pictures
  - the detailed product sheet will be displayed
    and you can access the original Open Food Facts page by clicking on "Voir la fiche d'Open Food Facts"
    or if you're logged in, you can add this product to your substitutes by clicking on "Ajouter à mes produits favoris"
    (otherwise, you'll be redirected to the sign in page)
  - access your saved products by clicking on the carrot logo (while you're logged in)
